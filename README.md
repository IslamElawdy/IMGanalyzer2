# AI Image Lab (KI-Bildlabor)

## English

an interactive application to **understand computer vision and AI** concepts. It is designed as a learning tool that explains how images are represented, processed, and interpreted by modern models.

### Features
1. **Upload & Info**: Image inspection and normalization.

2. **Pixel Explorer**: Interactive zoom and tensor values.
3. **Channels & Tensor**: RGB split and histograms.
4. **Patch Viewer**: Step-by-step convolution logic.
5. **Conv Playground**: Filters (Sobel, blur, etc.) on the full image.
6. **Activation & Pooling**: ReLU, MaxPool, and information loss.
7. **Noise & Robustness**: Distortions and JPEG artifacts.
8. **Augmentation**: Training data variations.
9. **Mini-CNN Features**: Insight into feature maps.
10. **Classification**: MobileNetV2 + Grad-CAM explainability.
11. **Adversarial Demo**: FGSM attack.
12. **Autoencoder**: Learn image compression.
13. **Training Dashboard**: Live training on synthetic data.

### Benefits
- **Hands-on learning** for computer vision and deep learning fundamentals.
- **Visual intuition** for tensors, convolutions, activations, and pooling.
- **Explainability** tools to understand model decisions.
- **Practical demos** of robustness and adversarial behavior.

### Installation
```bash
pip install -r requirements.txt
```

### Run
```bash
streamlit run app.py
```

## Deutsch

ist eine interaktive App zum **Verstehen von Computer Vision und KI**. Sie dient als Lernhilfe, um zu sehen, wie Bilder in modernen Modellen repräsentiert, verarbeitet und interpretiert werden.

### Features
1. **Upload & Info**: Bildanalyse und Normalisierung.
2. **Pixel Explorer**: Interaktiver Zoom und Tensor-Werte.
3. **Kanäle & Tensor**: RGB-Split und Histogramme.
4. **Patch Viewer**: Convolution-Logik Schritt für Schritt.
5. **Conv Playground**: Filter (Sobel, Blur, etc.) auf das ganze Bild.
6. **Activation & Pooling**: ReLU, MaxPool und Informationsverlust.
7. **Rauschen & Robustheit**: Störungen und JPEG-Artefakte.
8. **Augmentation**: Trainingsdaten-Variationen.
9. **Mini-CNN Features**: Einblick in Feature-Maps.
10. **Klassifikation**: MobileNetV2 + Grad-CAM Explainability.
11. **Adversarial Demo**: FGSM Attacke.
12. **Autoencoder**: Bildkompression lernen.
13. **Training Dashboard**: Live-Training auf synthetischen Daten.

### Vorteile
- **Praxisnahes Lernen** der Grundlagen von Computer Vision und Deep Learning.
- **Visuelle Intuition** für Tensoren, Convolutions, Aktivierungen und Pooling.
- **Erklärbarkeit** der Modellentscheidungen durch Grad-CAM.
- **Robustheits-Demos** inklusive Rauschen und adversarialen Beispielen.

## العربية


### الميزات
1. **رفع الصورة والمعلومات**: تحليل الصورة وتطبيعها.
2. **مستكشف البكسل**: تكبير تفاعلي وقيم الموتر.
3. **القنوات والموتر**: فصل RGB والمدرجات التكرارية.
4. **عارض الرقع**: منطق الالتفاف خطوة بخطوة.
5. **ساحة الالتفاف**: مرشحات (Sobel، Blur، إلخ) على الصورة كاملة.
6. **التفعيل والتجميع**: ReLU و MaxPool وفقدان المعلومات.
7. **الضوضاء والمتانة**: التشويش وآثار JPEG.
8. **تعزيز البيانات**: تنويعات بيانات التدريب.
9. **ميزات شبكة CNN صغيرة**: نظرة على خرائط السمات.
10. **التصنيف**: MobileNetV2 مع Grad-CAM لشرح القرار.
11. **عرض الخصومة**: هجوم FGSM.
12. **المشفّر التلقائي**: تعلّم ضغط الصور.
13. **لوحة التدريب**: تدريب مباشر على بيانات اصطناعية.

### الفوائد
- **تعلّم عملي** لأساسيات الرؤية الحاسوبية والتعلّم العميق.
- **فهم بصري** للموترات والالتفافات والتفعيل والتجميع.
- **أدوات تفسير** لفهم قرارات النموذج.
- **أمثلة تطبيقية** للمتانة والسلوك الخصومي.
